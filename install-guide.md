# DClaw 安装教程

> 通过 DGrid 一键快捷部署 DClaw，随时随地使用你的 AI 助手。

---

## 1. 查看你的 DGrid Pass

前往 [DGrid Pass 页面](https://premium.dgrid.ai/dgridpass)，登录后查看你持有的 DGrid Pass 列表。

## 2. 一键安装 DClaw

点击对应 DGrid Pass 的 **「一键安装 DClaw」** 按钮，系统将自动开始部署。

> **💡 提示：** 部署耗时取决于你的网络环境，通常需要数十分钟，请耐心等待。

## 3. 登录 Dashboard

安装完成后，点击 **「Dashboard」** 进入管理面板。

- **用户名 / 密码**：点击对应 DGrid Pass 的 **Secret Key** 即可查询你的 Dashboard 登录凭证。

## 4. 添加模型提供商

进入 Dashboard → **设置**，添加自定义模型提供商：

```
Base URL: https://api.dgrid.ai/v1/
```

## 5. 生成 API Key

前往 [DGrid API Keys](https://dgrid.ai/api-keys)，点击 **「创建新密钥」**，复制生成的密钥并粘贴至 Dashboard 模型提供商配置界面的 API Key 栏位。

## 6. 选择模型

前往 [DGrid 模型列表](https://dgrid.ai/models)，选择并复制你要使用的模型名称，在 Dashboard 中完成模型配置。

## 7. 开始使用 🎉

模型配置完成后，切换至 **Chat** 模块，即可开始畅玩 DClaw！

## 8. 进阶：接入飞书 / QQ

如需将 DClaw 接入飞书或 QQ，切换至 **频道** 页面，按照引导快捷完成渠道设置。
